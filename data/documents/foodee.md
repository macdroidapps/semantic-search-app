# Проект Foodee

## О проекте
Foodee - это веб-приложение для поиска и создания рецептов с AI-помощником.
Проект запущен в 2024 году.

## Разработчик
**Vladimir (blogsylar)** - Full-stack разработчик из Риги.
- GitHub: blogsylar
- Опыт: 5+ лет в разработке
- Специализация: Android (Kotlin), Web (Next.js)

## Технологии
- **Frontend:** Next.js 14 App Router
- **Styling:** Tailwind CSS
- **State:** Zustand
- **Backend:** Node.js + Express
- **Database:** PostgreSQL 15
- **Cache:** Redis
- **Deploy:** VPS на macdroid.ru

## Особенности
- AI-powered генерация рецептов
- Поиск по ингредиентам
- Персональные рекомендации
- Поддержка русского языка
- Мобильная версия

## Деплой
- **Домен:** foodee.fun
- **Сервер:** VPS Ubuntu
- **Process manager:** PM2
- **Web server:** Nginx (reverse proxy)
- **SSL:** Let's Encrypt

## Статус
Проект находится в активной разработке.
Планируется запуск публичной версии в Q1 2025.
