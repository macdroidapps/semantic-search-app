# История компании CloudTech Solutions

## Основание

Компания CloudTech Solutions была основана 1 сентября 2018 года в Санкт-Петербурге.

Основатели:
- Иван Смирнов (CEO) - бывший директор по продукту в Яндексе
- Мария Коваленко (CTO) - ex-Google Software Engineer
- Дмитрий Волков (CFO) - финансовый аналитик из McKinsey

Стартовый капитал: 10 миллионов рублей от angel инвесторов.

## Продукт

CloudTech разрабатывает платформу для управления облачной инфраструктурой.

Основные возможности:
- Мониторинг серверов в реальном времени
- Автоматическое масштабирование
- Управление затратами на облако
- Интеграция с AWS, Google Cloud, Azure
- AI-рекомендации по оптимизации

## Технологический стек

### Backend
- Go 1.21 для высоконагруженных сервисов
- Python 3.11 для ML и аналитики
- Node.js для API Gateway
- gRPC для межсервисного взаимодействия

### Базы данных
- PostgreSQL 15 (основная БД)
- MongoDB для логов
- Redis для кэширования
- ClickHouse для аналитики

### Инфраструктура
- Kubernetes для оркестрации
- Prometheus + Grafana для мониторинга
- ELK Stack для логов
- ArgoCD для CI/CD

### Frontend
- React 18 с TypeScript
- Next.js 14 для SSR
- Tailwind CSS для стилей
- Zustand для state management
