## Текущий проект

С января 2023 года Алексей возглавляет команду из 5 разработчиков. Они создают систему аналитики больших данных.

Проект использует современный стек:
- Backend: FastAPI + Python 3.11
- Database: PostgreSQL 15 с расширением TimescaleDB
- Message Queue: Apache Kafka
- Кэш: Redis 7
- Frontend: React 18 + TypeScript
- Deploy: AWS ECS с автомасштабированием

Бюджет проекта: 5 миллионов рублей.
Срок реализации: 12 месяцев.
